# Projet *OCRganiz*

## C'est quoi *OCRganiz* ?
Cela permet d'enregistrer des fichiers, telles que des factures, fiches de paie, en utilisant de l'OCR *(Opticial Recognization Character)* afin de pouvoir les trier.
